# shuurkhai.com

Shuurkhai - Америк барааг Монголд хурдан, найдвартай хүргэх карго үйлчилгээ.

## Төсөл

Энэ төсөл нь Shuurkhai.com вэбсайтын код юм. Америкийн онлайн дэлгүүрүүдээс (Amazon, Walmart гэх мэт) бараа захиалж, агаар болон далайн каргоор Монголд хүргүүлэх үйлчилгээ үзүүлдэг.

## Шаардлага

- PHP 7.4+
- MySQL/MariaDB
- Apache with mod_rewrite

## Суулгах

1. `config.example.php` файлыг хуулж `config.php` үүсгэх
2. `config.php` дотор database тохиргоогоо оруулах
3. `.htaccess` файл зөв тохируулагдсан эсэхийг шалгах

## Үндсэн функцүүд

- Шинэ нүүр хуудас (Tailwind CSS)
- Үнийн тооцоолуур (агаар/далайн карго)
- Онлайн захиалга
- Дэлгүүрүүдийн мэдээлэл
- Холбоо барих форм
