**Welcome to Shuurkhai Home Project**

**About**

This is the Shuurkhai home page project - a PHP-based website for cargo services from USA to Mongolia.

**Setup**

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies: `npm install`
4. Run the app: `npm run dev`

**Views**

The main home page is available at `views/home_new.php` which can be integrated into your PHP application.

**Tech Stack**

- React (for other pages)
- PHP (for home page)
- Tailwind CSS
- Vite
